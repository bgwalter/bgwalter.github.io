Data Structures in C - Fall 2014

N-body problem: Brute Force

For a final project, I wanted to create an implementation of the Barnes-Hut algorithm for the N-body problem. Unfortunately, I did not have time to finish this, but I was able to succesfully implement it via brute force.

This simulation was done in only two dimensions, and the gif on the right shows the results with 500 bodies.

Although the course itself was taught in C, I took the time to learn some C++ for this project. The full source code will be up on github.
