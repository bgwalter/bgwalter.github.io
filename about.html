I was born and raised in Seattle, Washington but am currently an undergraduate at Bennington College in Bennington, Vermont where I'm studying computer science and astrophysics.

This is a website I made, partly to re-teach myself HTML and CSS, but also to put my artwork and projects up onto as well as various references for myself and others to use. It is very much so an ever-on-going work in progress.

This site was written using Brackets on a computer running Arch Linux, which I use as a daily driver. The images on the site were created and edited using Krita and most of the digital art I make is made in it as well, although in the past I primarily used Adobe Photoshop.

The source code for this site is availble in full on github and is hosted using github Pages.
